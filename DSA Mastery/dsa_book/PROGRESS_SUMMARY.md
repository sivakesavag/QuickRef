## Key Concepts Covered So Far

### Mathematical Foundations
- Logarithms, exponents, modular arithmetic
- Combinatorics, probability
- Floor, ceiling, summation formulas
- Recurrence relations, Master Theorem
- Stirling's approximation, Akra-Bazzi method

### Complexity Analysis
- Big-O, Omega, Theta, Little-o, Little-omega
- Best, worst, average, amortized analysis
- P vs NP, NP-completeness basics
- Space complexity, cache complexity

### Recursion
- Base cases, recursive cases
- Recursion tree method
- Tail recursion, memoization vs tabulation
- Mutual recursion
- Trampolining, CPS, Y-combinator

### Linear Data Structures
- Arrays: static vs dynamic, cache locality, 2D arrays
- Linked Lists: singly, doubly, circular, skip lists
- Stacks: LIFO, expression evaluation, monotonic stacks
- Queues: FIFO, circular, deque, priority queues
- Hash Tables: chaining, open addressing, collision resolution

### Trees Fundamentals
- Binary tree properties and traversals
- Tree height, depth, diameter
- Complete, full, perfect binary trees
- Tree construction from traversals
- Morris traversal, threading
- LCA, tree serialization

### Binary Search Trees
- BST property, search, insert, delete
- In-order predecessor/successor
- BST validation, floor/ceiling
- Self-balancing motivation, degeneration

### Balanced Trees
- AVL trees: balance factor, rotations (LL, RR, LR, RL)
- Red-Black trees: properties, insertion fix-up, height bounds
- B-Trees and B+ Trees: multi-way search, disk optimization
- Splay trees: amortized O(log n), working set property
- Treaps: BST + heap hybrid, expected O(log n)

### Heaps & Priority Queues
- Binary heap: array representation, sift up/down
- Build heap O(n) proof, heap sort
- Priority queue ADT, median maintenance, K-way merge
- Fibonacci, binomial, leftist, skew, pairing heaps
- d-ary heaps, indexed priority queues

### Tries & String Data Structures
- Trie operations: insert, search, prefix matching
- Compressed trie (radix/Patricia), ternary search trie
- Suffix trie, suffix tree (Ukkonen's), suffix array + LCP
- Aho-Corasick multi-pattern matching
- XOR trie, FM-index, BWT, suffix automaton

### Graphs — Representation
- Adjacency matrix vs adjacency list vs edge list
- Weighted, directed, undirected, bipartite, DAG
- CSR format, graph density, implicit graphs
- Graph properties: degree, connectivity, planarity

### Disjoint Sets (Union-Find)
- Find and union operations, path compression
- Union by rank/size, inverse Ackermann α(n)
- Kruskal's MST, cycle detection, connectivity
- Weighted DSU, rollback DSU, small-to-large merging

### Segment Trees & Binary Indexed Trees
- BIT (Fenwick tree): prefix sums, lowbit navigation
- Segment tree: build, query, point update
- Lazy propagation for range updates
- Persistent segment tree, merge sort tree
- Sparse table for static RMQ, 2D BIT
- Li Chao tree, segment tree beats
